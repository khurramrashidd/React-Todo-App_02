# 📝 React Todo App (Day 2)

A simple and functional **Todo Application built using React** that allows users to manage daily tasks efficiently.

This project demonstrates handling dynamic lists, user input, and state updates in React.

This project is part of my **React learning journey (Day 2 Project)**.

---

## 🚀 Features

* Add new tasks
* Delete tasks
* Display dynamic task list
* Real-time UI updates
* Simple and clean interface

---

## 🛠 Built With

* **React (Vite)**
* JavaScript (ES6+)
* CSS

## ⚙️ Installation

### 1. Clone the repository

```bash
git clone https://github.com/yourusername/day-02-todo-app.git
```

### 2. Navigate into the folder

```bash
cd day-02-todo-app
```

### 3. Install dependencies

```bash
npm install
```

---

## ▶️ Usage

Run the development server:

```bash
npm run dev
```

Open the local server URL in your browser.
